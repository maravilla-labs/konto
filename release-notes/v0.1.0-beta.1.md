# First Beta Release of Maravilla Konto

This is the initial beta of Maravilla Konto — a desktop accounting application built for Swiss small businesses.

## Core Features

- **Accounting** — Swiss KMU chart of accounts, journal entries, bank reconciliation (CAMT.053), annual report (Jahresrechnung) PDF
- **Invoicing** — Invoice creation, recurring invoices, credit notes, dunning, Swiss QR-bill support (SCOR reference), PDF generation
- **CRM** — Contact management with company-person relationships, customer numbering
- **Project Management** — WBS hierarchy, budgets, rate functions, project numbering
- **Time Tracking** — Time entries with approval workflow, rounding, billable/non-billable
- **Expense Tracking** — Single expenses and employee expense reports
- **Payroll** — Swiss payroll with AHV/IV/EO, ALV, BVG, NBU, KTG, FAK, Lohnausweis PDF, pain.001 export
- **Multi-language** — UI and documents in English, German, French, and Italian
- **Command palette** — Quick navigation with Cmd+K / Ctrl+K

## Platform

- Desktop app for **macOS** and **Windows** (Tauri)
- Auto-update support via built-in updater
