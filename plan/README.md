# Hope (Maravilla Konto) - Open-Source Swiss Accounting
Hope is an open-source accounting tool built for Swiss SMEs. It provides full double-entry bookkeeping with Swiss compliance out of the box.

## Features

- Full double-entry accounting with Swiss KMU chart of accounts
- Invoice management (from time tracking, contacts, projects; recurring invoices)
    - Email and PDF generation
    - Layout designer for invoices
- Project management / Time tracking
- Customer relationship management (CRM)
- Document management
- Expense management / Tracking
- VAT management / Tracking
- Swiss employee management with payslips
- Bank reconciliation via ISO 20022 (CAMT.053) import
- Swiss QR-bill invoices
- User and role management
- Access for external auditors and accountants

## Production-Ready
- No shortcuts, no placeholders — fully functional
- Configurable (e.g. VAT) based on Swiss laws
- Multi-currency bank account management
- Dashboard with financial KPIs
- Employee time tracking access
