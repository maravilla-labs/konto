# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Maravilla Konto, please report it
responsibly. **Do not open a public GitHub issue.**

Send an email to **security@maravillalabs.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 5 business days
- **Fix or mitigation:** as soon as practical, depending on severity

## Scope

This policy covers the Maravilla Konto application code in this repository.
Third-party dependencies are out of scope but we appreciate being notified.

## Disclosure

We follow coordinated disclosure. Please allow us reasonable time to address
the issue before making it public.
